Let's integrate QuickCall with Claude Code.

We begin with installing MCP server.

We then install plug-in from the marketplace.

Once initial setup is done, we run QuickCall connect command. This command will generate a signup link.

Now complete the CLI authorization.

Once that is done, we connect GitHub and Slack.

Follow instructions on your browser to connect your repos to QuickCall.

Similarly, connect Slack to QuickCall.

You have now connected claude to QuickCall
